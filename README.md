This repository contains my personal configuration for **Neovim**
 and **Tmux** aswell as my config for sketchybar and Spicetify.

## Features

- **Transparent Neovim UI** that integrates cleanly with your terminal theme
- Extremly minimal TMUX TUI that integrates with the terminal theme
- Plugin-managed with TPM(tmux) and LazyVim

## Language Support

Pre-configured for:

- Java (with **Maven** and **Gradle**)
- C / C++
- Go
- TypeScript
- JavaScript
- Python
